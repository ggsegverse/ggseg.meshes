# Changelog

## ggseg.meshes 0.0.1

### New features

- Cortical fsaverage5 surfaces: pial, white, semi-inflated, sphere,
  smoothwm, orig
- SUIT cerebellar flatmap surface
- [`get_cortical_mesh()`](https://ggsegverse.github.io/ggseg.meshes/reference/get_cortical_mesh.md)
  for accessing cortical surfaces by hemisphere and type
- [`get_cerebellar_flatmap()`](https://ggsegverse.github.io/ggseg.meshes/reference/get_cerebellar_flatmap.md)
  for the SUIT flatmap mesh
- [`available_cortical_surfaces()`](https://ggsegverse.github.io/ggseg.meshes/reference/available_cortical_surfaces.md)
  and
  [`available_cerebellar_surfaces()`](https://ggsegverse.github.io/ggseg.meshes/reference/available_cerebellar_surfaces.md)
  for discovery
