# List available cortical surfaces

List available cortical surfaces

## Usage

``` r
available_cortical_surfaces()
```

## Value

Character vector of available surface names.

## Examples

``` r
available_cortical_surfaces()
#> [1] "pial"          "white"         "midthickness"  "semi-inflated"
#> [5] "sphere"        "smoothwm"      "orig"         
```
