# ggseg.meshes: Additional Brain Surface Meshes for the 'ggsegverse' Ecosystem

Provides additional brain surface meshes for cortical and cerebellar
visualisation in the 'ggseg' ecosystem. Cortical meshes include
'fsaverage5' pial, white, semi-inflated, sphere, smoothwm, and orig
surfaces. Cerebellar meshes include the 'SUIT' flatmap surface. All
meshes follow the same vertices/faces data frame format used by
'ggseg.formats' and 'ggseg3d'.
